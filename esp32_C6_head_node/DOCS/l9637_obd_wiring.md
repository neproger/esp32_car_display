# L9637D K-Line Wiring

Краткая схема подключения для связки:

- `ESP32-C6`
- `L9637D`
- `OBD K-line`

Подходит для проекта с `KWP1281` и `5-baud init`.

## ESP32 <-> L9637D

- `ESP32 GPIO16 (TX)` -> `L9637D TXD`
- `ESP32 GPIO17 (RX)` <- `L9637D RXD`
- `ESP32 GND` <-> `L9637D GND`

Примечание:

- `RXD` у `L9637D` может быть уровнем `5 V`, поэтому для входа `ESP32 RX` желательно поставить делитель напряжения.

Простой делитель:

- `L9637D RXD` -> `R1 10 kOhm` -> `ESP32 RX`
- `ESP32 RX` -> `R2 20 kOhm` -> `GND`

Такой делитель снижает уровень примерно до `3.3 V`.

## L9637D <-> OBD

- `L9637D K` -> `OBD pin 7` (`K-line`)
- `L9637D VS` -> `OBD pin 16` (`+12 V`)
- `L9637D GND` -> `OBD pin 4` или `OBD pin 5` (`GND`)

## Рекомендуемая защита и обвязка

### Обязательные конденсаторы

Ставить как можно ближе к микросхеме:

- `VCC` -> `100 nF` керамический -> `GND`
- `VS` -> `100 nF` керамический -> `GND`

Желательно дополнительно:

- `VS` -> `1 uF` керамический -> `GND`

### Резистор в линии K

Желательно:

- `L9637D K` -> `R3 510 Ohm` -> `OBD pin 7`

### TVS защита

Желательно:

- `K-line` -> `TVS 30 V` -> `GND`

Пример:

- `SMBJ30A`

## Минимальная итоговая схема

- `ESP32 GPIO16` -> `L9637D TXD`
- `L9637D RXD` -> `R1 10 kOhm` -> `ESP32 GPIO17`
- `ESP32 GPIO17` -> `R2 20 kOhm` -> `GND`
- `L9637D VS` -> `OBD pin 16 (+12 V)`
- `L9637D GND` -> `OBD pin 4/5 (GND)`
- `L9637D K` -> `R3 510 Ohm` -> `OBD pin 7 (K-line)`

## Список деталей

### Основные

- `U1`: `L9637D`, K-line transceiver, корпус `SO-8`

#### Аналоги
- `MC33660BEFR2`
- `MC33290`

### Резисторы

- `R1`: `10 kOhm`, `0.125 W` или `0.25 W`, `1%` или `5%`, выводной, металлопленочный
- `R2`: `20 kOhm`, `0.125 W` или `0.25 W`, `1%` или `5%`, выводной, металлопленочный
- `R3`: `510 Ohm`, `0.25 W`, `1%` или `5%`, выводной, металлопленочный

### Конденсаторы

- `C1`: `100 nF`, керамический, выводной, `50 V`, `X7R` или аналогичный, между `VCC` и `GND`
- `C2`: `100 nF`, керамический, выводной, `50 V`, `X7R` или аналогичный, между `VS` и `GND`
- `C3`: `1 uF`, керамический или пленочный, выводной, `25 V` или `50 V`, между `VS` и `GND`

### Защита

- `D1`: `TVS diode`, `SMBJ30A`, корпус `SMB`
- `D1 alternative`: выводной TVS или автомобильный супрессор на `30 V`, если нужен макет без SMD

## Практические замечания

- `5-baud init` реализуется программно на `ESP32`, аппаратно схема этому не мешает.
- Земля `ESP32`, `L9637D` и `OBD` должна быть общей.
- Обвязку питания и TVS лучше располагать максимально близко к `L9637D`.
- Для первого прототипа можно собрать минимальную схему без `TVS`, но для работы в автомобиле защита желательна.
- Если цель именно ручная пайка, пассивные элементы лучше брать выводные. `L9637D` останется SMD-компонентом, это нужно учитывать при сборке.
