# Golf 3 KWP1281: Блоки И Группы

Черновой справочник по адресам блоков управления и стартовым группам измерений для `Volkswagen Golf 3` с диагностикой `K-line / KWP1281`.

Важно:
- состав блоков зависит от года выпуска, рынка, двигателя и комплектации
- состав групп зависит от конкретного блока управления и его номера
- этот список подходит как стартовая карта для проекта, но не заменяет проверку на конкретной машине

## Основные Адреса Блоков

### Подтвержденные Типовые Адреса

- `01` - электроника двигателя (`Engine`)
- `02` - электроника АКПП (`Auto Trans`), если установлена `01M`
- `03` - ABS, если установлена
- `15` - Airbag, если установлен

### Адреса, Которые Нужно Проверять По Конкретной Машине

- `17` - приборная панель (`Instruments`)
- `25` - иммобилайзер (`Immobilizer`)

Примечание:
- на части `Golf 3` блок `17` может не поддерживаться так, как на более поздних VAG
- `25` может отсутствовать, быть отдельным блоком или быть связан с приборной панелью

## Блок 01: Двигатель

Для проекта это основной блок, с которого имеет смысл начинать опрос.

### Группа 001

Типовой набор:
- значение 1: обороты двигателя
- значение 2: температура охлаждающей жидкости
- значение 3: лямбда-коррекция
- значение 4: биты условий / базового режима

Статус:
- хороший кандидат для первого чтения
- часто содержит полезные базовые данные

### Группа 002

Типовой набор:
- значение 1: обороты двигателя
- значение 2: нагрузка
- значение 3: среднее время впрыска
- значение 4: расход воздуха или давление во впуске

Статус:
- использовать после проверки на конкретном ECU

Примечание:
- в разных системах последнее поле может отличаться

### Группа 003

Типовой набор:
- значение 1: обороты двигателя
- значение 2: расход воздуха или давление во впуске
- значение 3: угол дроссельной заслонки
- значение 4: угол опережения зажигания

Статус:
- полезная группа для двигателя
- часто используется для контроля воздуха, дросселя и зажигания

### Группа 004

Типовой набор:
- значение 1: обороты двигателя
- значение 2: напряжение питания
- значение 3: температура охлаждающей жидкости
- значение 4: температура воздуха на впуске

Статус:
- полезна для вывода питания и температур

Важно:
- ранее встречающиеся в интернете описания `group 004` как `idle/ac/load status` нельзя считать надежными без проверки по конкретному ECU

## Блок 02: АКПП 01M

Если на машине установлена `01M`, этот блок имеет смысл опрашивать отдельно.

### Группа 001

Часто ожидают:
- значение 1: обороты двигателя
- значение 2: скорость / входные обороты
- значение 3: скорость автомобиля
- значение 4: положение селектора

Статус:
- требует проверки на конкретном блоке

### Группа 002

Часто ожидают:
- значение 1: состояние соленоидов
- значение 2: текущая передача
- значение 3: статус блокировки гидротрансформатора

Статус:
- требует проверки на конкретном блоке

### Группа 005

Подтверждено как полезная сервисная группа:
- значение 1: температура масла в АКПП (`ATF temperature`)
- значение 2: напряжение питания
- значение 3: связанные служебные статусы

Статус:
- хороший кандидат для чтения на `01M`

## Блок 03: ABS

Наличие зависит от конкретной машины.

Типовые данные:
- скорости колес
- статусы клапанов
- сигналы датчиков ABS

Статус:
- нужен отдельный дамп или label file конкретного блока

## Блок 15: Airbag

Наличие зависит от комплектации.

Типовые данные:
- статусы системы
- ошибки
- кодирование

Статус:
- для проекта дисплея обычно не приоритетный блок

## Блок 17: Instruments

Нужно проверять отдельно.

Возможные данные:
- сервисные статусы
- кодирование
- отдельные диагностические значения панели

Статус:
- не считать гарантированно доступным для каждого `Golf 3`
